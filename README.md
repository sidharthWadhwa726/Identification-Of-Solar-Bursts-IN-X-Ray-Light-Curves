# Identification-Of-Solar-Bursts-IN-X-Ray-Light-Curves

## Usage
For the web interface,
- start a venv and install requirements.
- run server via `python backend/api/app`.
- head to web interface directory and set `.env` with `NEXT_PUBLIC_BACKEND_URI`. Set as `http://localhost:5000` for testing.
- install requirements with `npm i` and run dev server with `npm run dev`.
